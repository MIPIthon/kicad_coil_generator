**FILE:**

KICAD_COIL_GENERATOR.py

**SHORT DESCRIPTION**

- Script generates *.kicad_mod of a coil based on the user input provided in file.

- Functions available to create MxN matrices of coils


**GENERAL**

- Tested with KICAD 9.0.4
